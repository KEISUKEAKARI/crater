# スキル定義：ブランドマネージャー

## スキル名
**ブランドマネージャー** - ブランド名・要件を渡すだけでVI・Webイメージ・ブランドボードまで一気に設計する司令塔スキル

## 目的
名前だけ渡せばブランド全体をAIが設計する。
戦略・VI・コピー・Webイメージ・ブランドボードまで一気通貫。
複数案（デフォルト3案）を提示し、選んだ方向で本制作へ進む。

---

## インプットレベル（3段階対応）

| レベル | インプット例 | AIの動き |
|---|---|---|
| A. 名前だけ | `ミヤマグラフィックス` | 業種推定 → 戦略 → VI → Webイメージ → ボード |
| B. 要件あり | 業種・ターゲット・雰囲気など | 戦略から全自動 |
| C. 素材あり | 既存ロゴ・カラーなど参照 | VI整合 → Webイメージ → ボード |

---

## 起動時の動作

```
/ブランドマネージャー [ブランド名 or 要件]

例：/ブランドマネージャー ミヤマグラフィックス
例：/ブランドマネージャー 東京の美容室。30代女性向け。ナチュラルで上品。
例：/ブランドマネージャー output/brief.md を読んで
```

### 起動直後に必ず聞くこと（1問にまとめて）

引数の有無にかかわらず、**最初に必ずヒアリングシートの確認をする**：

```
【ブランドマネージャーです】

ブランドボードのヒアリングシートはありますか？

A. ある → Googleスプレッドシートを読みに行きます（何行目か教えてください）
B. ない → ブランド名・業種・キーワード3つを教えてください
```

- 「A. ある」と答えた場合：Googleドライブで「ブランドボード ヒアリングシート（回答）」を検索し、指定行のデータを取得してからStep 1へ進む
- 「B. ない」と答えた場合：引数があればそのまま、なければ3問（ブランド名・業種・キーワード3つ）を一括で聞いてStep 1へ進む
- 引数でフォルダパスが渡された場合は、そのフォルダの `CLAUDE.md` を読み込んだ上でヒアリングシート確認を行う

---

## フロー

### Step 1：ブランド分析・方向性3案を自動生成

インプットをもとにAIが3つのブランドディレクションを提示する。
各案にはセットで以下を含む：

```
【案A】[ディレクション名]
- 世界観：〜
- カラーイメージ：〜
- タイポグラフィイメージ：〜
- タグライン案：〜
- Webサイトの雰囲気：〜

【案B】[ディレクション名]
...

【案C】[ディレクション名]
...

→ どれかを選ぶ or「AとBをミックス」など指示してください。
```

### Step 2：選択後 → 自動で本制作へ

ユーザーが案を選んだら以下を順番に実行（確認不要・一気に進める）：

1. **ブランド戦略書** を生成（MVV・ターゲット・ポジショニング・ストーリー）
2. **`/ブランドボイス`** を呼び出してトーン・タグライン・コピー方針を生成
3. **`/VI設計者`** を呼び出してカラー・タイポ・デザインシステムを生成
4. **`/画像生成`** でWebサイトイメージ・キービジュアルをAI生成（nanobanana / ChatGPT Images）
5. **`/ブランドボード`** で上記を全部まとめたHTMLブランドボードを生成

### Step 3：完成報告

```
【ブランドボード完成】

📁 場所：[output/brand-board.html]
🎨 方向性：[選んだ案名]
✅ 含まれるもの：戦略 / VI / ボイス / Webイメージ

ブランド診断も実施しますか？（/ブランド診断）
```

---

## 複数案の生成ルール

- デフォルトは3案。「5案出して」など指定があれば対応
- 各案は方向性が明確に異なるものにする（似たものは出さない）
- 案の名前はキャッチーに（例：「静謐と余白」「遊び心と温度」「洗練と本質」）

---

## 出力先ルール

- 案件フォルダが渡された場合：`[案件フォルダ]/output/` に保存
- 渡されていない場合：ユーザーに保存先を確認する

---

## 禁止事項

- 1問ずつ聞かない（まとめて聞く）
- 確認が多すぎない（Step 1の案選択以外は自動で進める）
- 案を1つだけ出さない（最低2案）
- ブランドボードなしで終わらない（最終成果物は必ずHTML）

---

## 連携スキル

| スキル | 役割 |
|---|---|
| `/ブランドボイス` | トーン・コピー・タグライン設計 |
| `/VI設計者` | カラー・タイポ・デザインシステム生成 |
| `/画像生成` | WebイメージをAI生成 |
| `/ブランドボード` | 全要素をまとめたHTMLボード生成 |
| `/ブランド診断` | 既存ブランドの現状分析・改善提案 |
