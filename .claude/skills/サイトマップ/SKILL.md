# サイトマップ作成スキル

コーポレートサイトのページ構成を階層図（HTML）で作成します。

---

## 事前確認項目

以下を確認してからHTMLを作成する。

1. **サイト種別**：コーポレート / サービスサイト など
2. **目的**：ブランディング / 問い合わせ獲得 / 採用 など
3. **既存サイト**：あり（リニューアル）/ なし（新規）
4. **ページ構成**：どんなページが必要か（クライアントと合意済みのもの）

---

## HTML構成のデザインルール

### レイアウト
- TOPページを頂点に、ページを横並びで展開する階層図
- 横線は `.pages::before` 擬似要素で描く（はみ出し防止）
- 固定幅（px）は使わず `width: 100%` + `max-width` で可変にする（スマホ対応）

### カラー
- TOPボックス背景：`#1e2d3d`（ダークネイビー）
- ページボックス：白背景 + `#1e2d3d` ボーダー
- サブページ：`#f9f9f9` 背景 + `#ccc` ボーダー
- 線：`#bbb`

### フォント
- `'Helvetica Neue', Arial, 'Hiragino Sans', sans-serif`

### スマホ対応の注意点
- 横線は固定幅にしない → `.pages::before` で `left: 12.5%` `right: 12.5%` で制御
- ページボックスの幅は `width: 90%; max-width: 140px`
- サブボックスの幅は `width: 90%; max-width: 130px`

## 固定ルール
- お問い合わせページがある場合：

  - 「プライバシーポリシー」はお問い合わせと同列の独立ページとして追加する
