# レスポンシブ確認

プレビューサーバーを起動し、以下の3サイズでスクリーンショットを撮って見せてください。

## 手順

1. 対象サイトのプレビューサーバーを起動する（$ARGUMENTS が指定されていればそのサーバー名を使う）
2. 以下の3サイズそれぞれでスクリーンショットを撮る：
   - **スマホ**（375×812）
   - **タブレット**（768×1024）
   - **PC**（1280×800）
3. 各スクリーンショットをユーザーに見せる
4. 表示崩れや気になる点があればコメントする

## ルール

- スクリーンショットは必ず3サイズすべて撮ること
- 各サイズの名前（スマホ・タブレット・PC）を明記すること
- 問題がなければ「問題なし」と伝えること
