# レイアウト指示ツール

LP や Web サイトのセクションレイアウトを直感的に設計・指示するツール。

## 機能

✅ **セクション選択**：右側のパレットからセクションを選択
✅ **ドラッグ&ドロップ**：セクションをキャンバスにドラッグして配置
✅ **パターン選択**：各セクションで複数のレイアウトパターンから選択
✅ **カスタムセクション**：プロジェクトに必要なセクションを追加可能
✅ **画像出力**：完成したレイアウトを PNG で出力
✅ **JSON 出力**：セクション構成を JSON で出力（Claude への指示用）

---

## 使い方

### 1. ツール起動

```bash
python3 -m http.server 8084 -d ./layout-instruction-tool
```

または Claude Code で：

```
/preview-start layout-instruction-tool
```

ブラウザで `http://localhost:8084` を開く

### 2. セクション追加

- 右側の **セクション選択パレット** からセクションをドラッグ
- 左側の **レイアウトキャンバス** にドロップ
- セクションが追加される

### 3. パターン選択

- 各セクションの **「パターン変更」** ボタンをクリック
- 複数のレイアウトパターンから選択
- 選択すると右側にプレビュー表示

### 4. セクション削除・並び替え

- **「削除」** ボタンでセクションを削除
- ドラッグ&ドロップで並び替え可能（未実装）

### 5. 出力

- **「画像出力」**：レイアウト設計を PNG で出力
- **「JSON 出力」**：セクション構成を JSON で出力

## セクションタイプ（デフォルト）

### FV（ファーストビュー）
- 左テキスト＋右ビジュアル
- 中央テキスト＋背景
- フルスクリーンビジュアル

### テキストメイン
- シンプルテキスト
- リスト＋画像
- セクション見出し

### カード配置
- 3 列グリッド
- 2 列グリッド
- 1 列（フル幅）

### テーブル/比較表
- 標準テーブル
- 比較表（カード形式）
- FAQ（アコーディオン）

### コンタクト
- シンプルフォーム
- マルチカラムフォーム
- チェックボックス多数

### ブログ/ニュース
- サムネイル＋テキスト
- グリッド表示（3 列）
- リスト表示

---

## カスタムセクション追加

1. **「+ 追加」** ボタンをクリック
2. セクション名を入力（例：「スライダー」「タイムライン」）
3. 説明とカラー（識別用）を指定
4. **「作成」** をクリック

カスタムセクションは右側のパレットに追加され、デフォルトセクションと同じように使用可能。

---

## 次のステップ

出力した JSON または画像をもとに：

1. **Figma でレイアウトフレーム作成**
   - セクション構成をベースにレイアウト詳細化
   - パディング・マージン・高さなど具体値を決定

2. **Claude への指示**
   - JSON を Claude に読み込ませてレイアウトフレーム制作指示
