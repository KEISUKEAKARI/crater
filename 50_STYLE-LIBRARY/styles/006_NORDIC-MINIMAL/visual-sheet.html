Do / Don't
Do
  • warm-white背景（#FAFAF8）に統一する
  • sage緑は差し色に小量使用。面積を広げない
  • DM Serif Displayのイタリックを効果的に使う
  • 余白は広く。素材と空間を呼吸させる
  • 機能的なシンプルさを最優先にする
Don't
  • 冷たい青白い色調（純粋な白・クールグレー）
  • 多色使い（アクセントはsage1色のみ）
  • 装飾的な書体・手書き系フォント
  • 過剰なアニメーション・動きの多い演出
  • ビビッドカラーとの組み合わせ

Layout Rules
Max Width
コンテンツ幅は 900px を上限。テキストブロックは 640px。
Column Padding
左右余白は 64px。モバイルは 32px。余白を削らない。
Line Height
本文は 1.85〜1.9。見出しは 1.1〜1.3。呼吸する間を取る。
Section Spacing
セクション間は 64〜96px。余白が品格をつくる。
Color Use
sageは全体の 10%以下。基調はcharcoal × warm-white のみ。
Italic Rule
DM Serif Italicは H2のみ に使用。多用すると効果が薄れる。
