Do / Don't
Do
  • Bebas Neue ALL CAPS で強打する。中途半端な大きさは禁止
  • 余白をリズムとして意図的に使う。詰めても抜いても計算する
  • イエローは面積で使える。背景ごと染めてよい
  • 2色のコントラストを最大化。黒+黄の組み合わせを核に
  • レッドはアクセント用。1ページに1〜2箇所まで
Don't
  • グラデーション（単色の強さを希薄にする）
  • 角丸（border-radius）。エッジはシャープに保つ
  • 細いフォント・軽い書体（Thin / Light 系）
  • パステルカラーや中間色。存在感が消える
  • センタリングの多用。左揃えを基本に組む

Layout Rules
Max Width
コンテンツ幅は 1080px を上限。ヒーローはフルブリード可。
Column Padding
左右パディング 64px（モバイル 24px）。
Border Radius
全要素 0px。角丸は一切使わない。シャープエッジが原則。
Typography Grid
テキストは左揃え基本。Bebas Neueは字間 0.01em で組む。
Section Spacing
セクション間は 64–96px。詰めてもリズムを崩さない。
Color Usage
イエローは面積で使える。レッドは 1–2箇所 まで。グラデーション禁止。
