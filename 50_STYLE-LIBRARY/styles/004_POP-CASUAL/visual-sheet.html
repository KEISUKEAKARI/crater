Do / Don't
Do
  • 丸みのあるborder-radius（8px〜pill形）を積極活用
  • コーラル・イエロー・ティールを均等に使う
  • Nunitoの太めウェイト（800・900）を見出しに
  • 重さを感じさせない。フラットでポップな印象を保つ
  • ボタンは大きめにして押しやすくする
Don't
  • 黒背景・重いダークトーン
  • シリアスなトーン・権威的な言葉遣い
  • 細い線・繊細すぎる装飾
  • 過剰なシャドウ・リアルな立体表現
  • 小さい文字・読みにくいコントラスト

Layout Rules
Max Width
コンテンツ幅は 900px を上限。セクションによって 720px に絞る場合あり。
Column Padding
左右余白は 64px。モバイルは 32px。
Border Radius
カードは 16px。ボタンは 999px（pill）。直角は使わない。
Section Spacing
セクション間は 64px（--sp-8）。ポップだからといって詰め込まない。
Color Balance
3色（コーラル・イエロー・ティール）は 均等 に使う。1色偏重は禁止。
Font Weight
見出し最低 700。本文は 400。細すぎる文字は使わない。
