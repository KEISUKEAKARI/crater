Do / Don't
Do
  • ダーク背景（#141414 / #1E1E2E）に統一する
  • Space MonoをラベルとコードUIに徹底使用
  • エレクトリックブルーは差し色のみ・面積を抑える
  • グリッドを守る。要素を揃える
  • 余白は広く。情報密度は意図的にコントロール
Don't
  • ウォームカラー・アースカラーの使用
  • 過剰な装飾・グラデーション・シャドウ多用
  • 多色使い（アクセント色は1色に限定）
  • 丸みが大きいボーダーラジウス（8px以上禁止）
  • 手書き・スクリプト系フォント

Layout Rules
Max Width
コンテンツ幅は 900px を上限とする。それ以上は左右に余白。
Column Padding
左右余白は 64px。モバイルは 24px。
Grid System
12カラムグリッド。ガターは 24px。要素は必ずグリッドに揃える。
Section Spacing
セクション間は 64px（--sp-8）。コンテンツ密度を一定に保つ。
Border Radius
最大 4px。丸みは機能的にのみ使用。装飾目的では使わない。
Line Height
見出しは 1.1〜1.25。本文は 1.7〜1.8。密度設計を意識する。
