素材感のアップ
色温度：5000K前後の昼白色。青みは少し残してもよい。
露出：やや明るめ。ハイライトを飛ばさず、やさしく仕上げる。
構図：中央か上寄り。余白を多く取り、空気感を大切にする。
Do / Don't
Do
  • 丸みのある border-radius（8〜999px）を積極的に使う
  • パステル3色の調和。ブラッシュ・ラベンダー・クリームが軸
  • Playfair Display italic を効果的なアクセントとして
  • 余白を多め。詰め込まず、空気感を残す
  • やわらかいシャドウ（box-shadow）で立体感を出す
Don't
  • ビビッドカラーや彩度の高い色の使用
  • 重いシャドウやハードな影
  • シャープコーナー（border-radius: 0）
  • マスキュリンな構成・無骨な書体
  • ダークバックグラウンドへの展開（原則クリーム系）

Layout Rules
Max Width
コンテンツ幅は 1080px を上限。余白を惜しまない。
Column Padding
左右パディング 64px（モバイル 32px）。
Border Radius
カード 16px、ボタン 999px。角丸は積極的に使う。
Shadow
シャドウは blur 24px 以上。やわらかく・軽く。ハードシャドウ禁止。
Section Spacing
セクション間は 96–128px。空気感を大切に。
Typography
見出しは Playfair italic。Inter は weight 300 を基本に組む。
