Do / Don't
Do
  • グラデーション（purple→pink）をボタン・装飾に積極的に使う
  • glow効果（box-shadow でのネオン表現）を要所で使う
  • Syne の太字（weight 800）でタイトルを打ち抜く
  • 背景は暗く保つ。void・dark-purple が基調
  • シアンをテキストアクセントとして点在させる
Don't
  • ウォームカラー（オレンジ・ベージュ・ブラウン系）
  • プリント的な質感・紙の風合いの表現
  • serif書体（アナログ感が出る）
  • 明るい背景・白ベースのレイアウト
  • グロー効果なしの素朴なカラー使用

Layout Rules
Max Width
コンテンツ幅 1080px。ヒーローはフルブリード。
Column Padding
左右パディング 64px（モバイル 24px）。
Background
ベースは void (#050505)。白や明るい背景は使わない。
Glow Usage
box-shadow のグローは 主要CTAと見出し のみ。多用しない。
Section Spacing
セクション間は 96–128px。暗い背景に余白が映える。
Gradient Rule
グラデーションは purple → pink が基本軸。ランダムな配色禁止。
