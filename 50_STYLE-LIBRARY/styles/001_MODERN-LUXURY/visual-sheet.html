ひとつだけ置く
色温度：3200K前後のウォームトーン。青みは排除する。

Do / Don't
Do
  • 余白は惜しまない。コンテンツを詰め込まない
  • 金は「差し色」として使う。面積を大きくしない
  • フォントはCormorantとInterの2種のみ
  • テキストは少なく、選び抜いた言葉だけ使う
  • グラデーションは黒〜透明のみ許容
Don't
  • 派手なアニメーションやトランジション
  • ビビッドカラーとの組み合わせ
  • フォントを3種以上使う
  • 影を多用する（flatかわずかなシャドウのみ）
  • 丸みの大きいボーダーラジウス（2px以上は禁止）

Layout Rules
Max Width
コンテンツ幅は 1080px を上限とする。ヒーローセクションはフルワイド可。
Column Padding
左右パディングは 64px（モバイル 24px）。
Grid
12カラムグリッド基準。gap 24px。コンテンツは4・6・8カラム単位で組む。
Section Spacing
セクション間の余白は 96–128px。詰めすぎない。
Border Radius
最大 4px。角丸は最小限。シャープなエッジがこのスタイルの原則。
Motion
トランジションは 0.15s ease のみ使用。派手なアニメーション禁止。
