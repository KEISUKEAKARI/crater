Do / Don't
Do
  • 行間は2.0以上。文字を詰め込まない
  • 余白を惜しまない。空間はメッセージ
  • 漢字と欧文の混植を活かす
  • 朱色（--c-red）はポイントとして一点に使う
  • Noto Serif JPのweightは300〜400のみ使用
Don't
  • 過剰な装飾・幾何学パターン
  • ビビッドカラー・高彩度の色
  • 丸みの大きいborder-radius（4px以上禁止）
  • 多色使い（3色以上は禁止）
  • ゴシック体・サンセリフを見出しに使う

Layout Rules
Max Width
コンテンツ幅は 900px を上限。テキストは 640px に絞る。
Column Padding
左右余白は 64px。余白は惜しまない。
Line Height
本文は最低 2.0。見出しは 1.5〜1.8。詰めない。
Section Spacing
セクション間は 96px（--sp-9）。余白が世界観をつくる。
Border Radius
最大 2px。基本的に使用しない。直角が和の緊張感を保つ。
Letter Spacing
見出しは +0.05〜+0.08em。ゆったりとした呼吸。
